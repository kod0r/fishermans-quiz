## Workflow rules for this cycle

1. **Branch:** `dev-refactor` only. No sub-branches.
2. **Commit format:** `fix(scope): description (#<issue>)`
3. **Pre-commit hooks active:** `lint-staged → tsc -b → vitest run`
4. **If hook fails:** fix it. Never `--no-verify` except true emergencies.
5. **Issue lifecycle:**
   - Code complete → add `pending-review` label + comment with summary
   - Leave issue **open**
   - User closes after review
